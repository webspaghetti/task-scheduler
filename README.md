## Overview
This repository is a monorepo containing both the frontend and backend applications for the Task Scheduler platform. The system is designed to handle user authentication, team organization, and task management securely and efficiently.

>**Note:** This project was created as a learning exercise for Spring Boot, focusing on building a REST API and understanding backend architecture in a full-stack environment.

##  Project Structure
* [**`/scheduler-server`**](./scheduler-server/README.md): The Spring Boot Java backend (REST API).
* [**`/scheduler-client`**](./scheduler-client/README.md): The Next.js frontend application.
