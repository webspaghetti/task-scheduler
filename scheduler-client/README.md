The frontend interface for the Task Scheduler, built to be fast, responsive, and intuitive.

## Tech Stack
* Next.js (App Router)
* TypeScript
* Tailwind CSS

## Getting Started

1. Install dependencies:<br>
   ```bash
   npm install
   ```
   

2. Set environment variables (Create a .env.local file):<br>
   ```
   NEXT_PUBLIC_API_URL=http://localhost:8080
   ```


3. Run the development server:<br>
   ```bash
   npm run dev
   ```


Open http://localhost:3000 in your browser to view the application.
